📌 O que são Listas, Tuplas e Dicionários?

São estruturas que permitem guardar muitos valores dentro de uma única variável!


  • TUPLA = usa ( ) e não pode mudar (imutável)
  • LISTA = usa [ ] e pode mudar (mutável)
  • DICIONÁRIO = usa { } e guarda pares de CHAVE:VALOR
💡 Dica importante: Sem essas estruturas, você teria que criar uma variável para cada aluno, cada produto, cada signo... seria um trabalho enorme!

📦 Tuplas - Não podem ser alteradas

Tuplas usam parênteses ( ) e são IMUTÁVEIS (depois de criadas, não mudam).

Exemplo: Criando e acessando uma tupla

📄 tupla_simples.py 
# Criando uma tupla com as estacoes do ano
estacoes = ("Primavera", "Verao", "Outono", "Inverno")

# Exibindo a tupla completa
print(estacoes)

# Acessando elementos por indice (comeca em 0)
print(f"Primeira estacao: {estacoes[0]}")
print(f"Ultima estacao: {estacoes[-1]}")

# Verificando o tipo da variavel
print(f"Tipo: {type(estacoes)}")
⚠️ Importante: Se você tentar alterar um valor da tupla (ex: estacoes[0] = "Verao") o Python vai dar erro!

📋 Listas - Podem ser alteradas

Listas usam colchetes [ ] e são MUTÁVEIS (você pode alterar, adicionar ou remover).

Exemplo 1: Criando e acessando uma lista

📄 lista_criar.py 
# Criando uma lista de alunos
alunos = ["Ana", "Bruno", "Carla", "Daniel"]

# Exibindo a lista completa
print(f"Lista de alunos: {alunos}")

# Acessando o primeiro elemento (indice 0)
print(f"Primeiro aluno: {alunos[0]}")

# Verificando quantos elementos tem na lista
print(f"Quantidade de alunos: {len(alunos)}")

Exemplo 2: Alterando e adicionando elementos

📄 lista_alterar_adicionar.py 
# Lista original
alunos = ["Ana", "Bruno", "Carla"]
print(f"Original: {alunos}")

# ALTERAR: troca o elemento do indice 1 (Bruno) por Beatriz
alunos[1] = "Beatriz"
print(f"Após alterar: {alunos}")

# ADICIONAR: insere um novo elemento no final da lista
alunos.append("Daniel")
print(f"Após adicionar: {alunos}")

Exemplo 3: Removendo elementos (EXCLUSÃO)

📄 lista_remover.py 
# Lista original com 5 alunos
alunos = ["Ana", "Bruno", "Carla", "Daniel", "Elisa"]
print(f"Original: {alunos}")

# METODO 1: remove() - remove pelo VALOR (primeira ocorrencia)
alunos.remove("Carla")
print(f"Após remove(Carla): {alunos}")

# METODO 2: pop() - remove pelo INDICE e RETORNA o valor removido
removido = alunos.pop(2)
print(f"Após pop(2): {alunos}")
print(f"Elemento removido: {removido}")

# METODO 3: del - remove pelo INDICE (nao retorna o valor)
del alunos[0]
print(f"Após del alunos[0]: {alunos}")

Exemplo 4: Ordenando uma lista

📄 lista_ordenar.py 
# Lista de numeros desordenada
numeros = [5, 2, 8, 1, 9]
print(f"Original: {numeros}")

# sort() - ordena em ordem CRESCENTE
numeros.sort()
print(f"Ordenado (crescente): {numeros}")

# sort(reverse=True) - ordena em ordem DECRESCENTE
numeros.sort(reverse=True)
print(f"Ordenado (decrescente): {numeros}")

Exemplo 5: Soma, maior e menor valor

📄 lista_numeros.py 
# Lista de valores numericos
valores = [10, 25, 30, 15, 40]
print(f"Lista: {valores}")

# sum() - soma todos os elementos
print(f"Soma: {sum(valores)}")

# max() - encontra o maior valor
print(f"Maior valor: {max(valores)}")

# min() - encontra o menor valor
print(f"Menor valor: {min(valores)}")
💡 Resumo dos métodos de exclusão:
• remove(valor) - remove pelo VALOR
• pop(índice) - remove pelo ÍNDICE e retorna o valor
• del lista[índice] - remove pelo ÍNDICE (não retorna o valor)

📚 Dicionários - Pares de Chave:Valor

Dicionários usam chaves { } e acessam valores por uma CHAVE (não por índice).

Exemplo 1: Criando um dicionário

📄 dicionario_criar.py 
# Criando um dicionario com informacoes de uma pessoa
pessoa = {
    "nome": "Joao",
    "idade": 25,
    "cidade": "Sao Paulo"
}

print(pessoa)
print(f"Tipo: {type(pessoa)}")

Exemplo 2: Acessando valores pelas chaves

📄 dicionario_acessar.py 
# Dicionario da pessoa
pessoa = {"nome": "Joao", "idade": 25, "cidade": "Sao Paulo"}

# Acessando valores usando as CHAVES (nao usa indice!)
print(f"Nome: {pessoa['nome']}")
print(f"Idade: {pessoa['idade']}")
print(f"Cidade: {pessoa['cidade']}")

Exemplo 3: Alterando e adicionando valores

📄 dicionario_alterar_adicionar.py 
# Dicionario original
pessoa = {"nome": "Joao", "idade": 25, "cidade": "Sao Paulo"}
print(f"Original: {pessoa}")

# ALTERAR: muda o valor da chave "idade"
pessoa["idade"] = 26
print(f"Após alterar idade: {pessoa}")

# ADICIONAR: cria uma nova chave "profissao"
pessoa["profissao"] = "Engenheiro"
print(f"Após adicionar profissao: {pessoa}")

Exemplo 4: Removendo chaves (EXCLUSÃO)

📄 dicionario_remover.py 
# Dicionario completo
pessoa = {
    "nome": "Joao",
    "idade": 25,
    "cidade": "Sao Paulo",
    "profissao": "Engenheiro",
    "telefone": "99999-9999"
}
print(f"Original: {pessoa}")

# METODO 1: del - remove a chave (nao retorna o valor)
del pessoa["profissao"]
print(f"Após del pessoa[profissao]: {pessoa}")

# METODO 2: pop() - remove a chave e RETORNA o valor
telefone_removido = pessoa.pop("telefone")
print(f"Após pop(telefone): {pessoa}")
print(f"Telefone removido: {telefone_removido}")

# METODO 3: popitem() - remove e retorna o ULTIMO par
chave_removida, valor_removido = pessoa.popitem()
print(f"Após popitem(): {pessoa}")
print(f"Ultimo par removido: {chave_removida}:{valor_removido}")

Exemplo 5: Percorrendo um dicionário com for

📄 dicionario_for.py 
# Dicionario da pessoa
pessoa = {
    "nome": "Joao",
    "idade": 25,
    "cidade": "Sao Paulo",
    "profissao": "Engenheiro"
}

print("Informacoes da pessoa:")
for chave, valor in pessoa.items():
    print(f"{chave}: {valor}")
💡 Resumo dos métodos de exclusão em dicionários:
• del dicionario["chave"] - remove a chave (não retorna o valor)
• pop("chave") - remove a chave e RETORNA o valor
• popitem() - remove e retorna o ÚLTIMO par chave:valor
